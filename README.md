## 🌟 What is AeI?

**AeI** is a **smart Android AI chatbot** built entirely in **Pure Kotlin**, designed for
**students and developers** who want to test and interact with their own AI models
*(LM Studio, NVIDIA, Google, and more!)* — with features that even
**ChatGPT and Gemini don't have!**

> 💡 *AeI is not just another chatbot. It's your personal AI playground.*

---

## ✨ Key Features

| Feature | Description |
|--------|-------------|
| 🪄 **Prompt Enchanter** | Automatically enhances your prompts for better AI responses — *unique to AeI!* |
| 📱 **Open Apps on Device** | Ask AeI to open apps directly on your Android device — *no other AI chat app does this!* |
| 🔍 **Web Search** | Integrated **SearXNG** support for real-time web search results |
| 🌐 **Ngrok Compatibility** | Connect to your local AI server using **Ngrok's free tier** — no paid plan needed! |
| 🗣️ **5 Languages Support** | Communicate in **5 different languages** comfortably |
| 🎨 **Cool Design** | A sleek, modern and beautifully styled interface |
| 🧪 **Beta Features** | Access cutting-edge experimental features before anyone else |

## 📲 Installation

Getting started with AeI is super simple!

```
1. Go to the latest GitHub Release
2. Download the APK file
3. Install it on your Android device
4. Connect your AI backend & enjoy!
```

> ⚠️ **Note:** You may need to enable
> *"Install from Unknown Sources"* in your Android settings.

[![Download Latest Release](https://img.shields.io/badge/Download-Latest%20Release-brightgreen?style=for-the-badge&logo=android)](https://github.com/ArtemYatsyuk/AeI-Chatbo/releases/latest)

---

## 🚀 Quick Start with Ngrok

1. Install **Ngrok** and create a free account at [ngrok.com](https://ngrok.com)
2. Run your local AI model *(e.g. LM Studio)*
3. Expose it with Ngrok:

```bash
ngrok http 1234
```

4. Copy the **Ngrok URL** *(e.g. https://xxxx.ngrok-free.app)*
5. Paste it into AeI's server settings and start chatting! 🎉
